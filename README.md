# Glow Beauty Salon Website

A modern, responsive landing page for a beauty salon built with React, Vite, and Tailwind CSS.

## Features

- Responsive navbar with centered logo and split navigation links
- Hero section with branding and call-to-action
- Services cards with pricing and descriptions
- Customer reviews section
- About section with social links
- Floating WhatsApp contact button for quick appointment inquiries

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Icons

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Deployment

This project can be deployed easily on Netlify, Vercel, or any other static hosting platform.

## Notes

- Update the WhatsApp number in the app button configuration if needed.
- Replace branding assets such as the logo and hero image in the src/assets folder to match your salon identity.
